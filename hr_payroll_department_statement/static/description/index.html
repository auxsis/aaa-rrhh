HR Payroll Department Statement Excel

This module allow to print department wise payroll summary in excel format.
Depends on https://pypi.python.org/pypi/xlwt - So you have to install on server.

Menus:

  • Payroll
    • Report
      • Payroll Department Statement

Live Demo :- https://youtu.be/jRNqfNezSbM

List of Employee PaySlips


Payroll Department Statement Wizard


Menu to open to print payroll slip statement from wizard

By clicking this Export Excel button it will export department wise payroll statement for selected department. Note that department on wizard is option if you do not select department in wizard then module will print for all department in system where payslips have been generated. If department is not available on employee form then it will group all those payslips in Unknow department and show in excel as well.
