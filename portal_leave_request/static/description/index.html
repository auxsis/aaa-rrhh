Overview

The module allows both portal user and internal user to manage their leaves from the website. Employees can request their leaves also cancel their leaves using the application. Requested leaves can be viewed with alongside their status.

Features

  • Requesting leaves from website.
  • A corresponding leave request will be created in the backend.
  • Created leave requests can be viewed from the website.
  • An option to cancel the leave requests

Screenshots
